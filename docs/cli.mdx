# CLI Reference

This page documents all command-line options for QA Agent.

## Command Syntax

```bash
qa-agent [options]
```

## Core Options

| Option | Shorthand | Description | Default |
| ------ | --------- | ----------- | ------- |
| `--repo-path` | `-r` | Path to the repository | Required |
| `--config` | `-c` | Path to configuration file | None |
| `--verbose` | `-v` | Enable verbose logging | False |
| `--output` | `-o` | Output directory for generated tests | `./generated_tests` |
| `--target-coverage` | `-t` | Target coverage percentage | 80.0 |

## Parallel Processing Options

| Option | Shorthand | Description | Default |
| ------ | --------- | ----------- | ------- |
| `--parallel` | `-p` | Enable parallel test generation and validation | False |
| `--max-workers` | `-w` | Maximum number of workers for parallel execution | 4 |
| `--incremental` | `-i` | Only test files that have changed recently | False |
| `--changed-since` | None | Only test files changed in the last N days | 7 |

## Sourcegraph Integration

| Option | Description | Default |
| ------ | ----------- | ------- |
| `--enable-sourcegraph` | Enable Sourcegraph integration | False |
| `--sourcegraph-endpoint` | Sourcegraph API endpoint | `https://sourcegraph.com/.api` |
| `--sourcegraph-token` | Sourcegraph API token | None |

## AI Provider Options

| Option | Description | Default |
| ------ | ----------- | ------- |
| `--model-provider` | AI model provider (openai, anthropic, github-copilot) | `openai` |
| `--model-name` | Name of the model to use | `gpt-4o` |

## IP Protection Options

| Option | Description | Default |
| ------ | ----------- | ------- |
| `--ip-protection` | Enable IP protection | False |
| `--ip-rules-path` | Path to IP protection rules file | None |

## Examples

### Basic Usage

```bash
qa-agent --repo-path /path/to/your/project
```

### Using a Configuration File

```bash
qa-agent --repo-path /path/to/your/project --config ./qa_config.yaml
```

### Parallel Processing with Incremental Testing

```bash
qa-agent --repo-path /path/to/your/project --parallel --max-workers 8 --incremental --changed-since 3
```

### Using Anthropic's Claude Model

```bash
qa-agent --repo-path /path/to/your/project --model-provider anthropic --model-name claude-3-opus-20240229
```

### With Sourcegraph Integration and IP Protection

```bash
qa-agent --repo-path /path/to/your/project --enable-sourcegraph --sourcegraph-token your-token --ip-protection
```

### Setting Custom Coverage Target

```bash
qa-agent --repo-path /path/to/your/project --target-coverage 90.0
```

## Environment Variables

QA Agent recognises these environment variables:

| Variable | Description |
| -------- | ----------- |
| `OPENAI_API_KEY` | API key for OpenAI models |
| `ANTHROPIC_API_KEY` | API key for Anthropic models |
| `GITHUB_COPILOT_API_KEY` | API key for GitHub Copilot |
| `SOURCEGRAPH_API_TOKEN` | API token for Sourcegraph integration |

## Exit Codes

| Code | Description |
| ---- | ----------- |
| 0 | Successful execution |
| 1 | General error |
| 2 | Configuration error |
| 3 | API key error |
